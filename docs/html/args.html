predefinedPrepost

The predefined prefixes and postfixes.

  • $$ -- Default when no comment like Markdown
  • <!--$ and --> -- HTML
  • #$ -- Bash, python, etc.
  • ;$ -- Config files, Lisp
  • //$ -- C++
  • /*$ and */ -- C, C++
  • &lt;!--$ and --&gt; -- HTML inside a textarea element
  • # $ -- Org Mode
