Index

  • args.nim — Types for handling command line arguments.
  • cmdline.nim — Parse the command line.
  • comparelines.nim — Compare lines of text.
  • env.nim — Environment holding the input and output streams.
  • functions.nim — This module contains the StaticTea functions and supporting types.
  • gethelp.nim — Get the command line help message.
  • linebuffer.nim — Read lines from a stream without exceeding the maximum line length.
  • matches.nim — Methods for matching sub-strings.
  • messages.nim — Messages IDs and associated strings and routines to get them.
  • opresult.nim — OpResult holds either a value or a message.
  • parseCmdLine.nim — Parse a StaticTea language command line.
  • parseCommandLine.nim — Parse the StaticTea terminal command line and return the arguments.
  • parseMarkdown.nim — Parse the simple markdown used in the function descriptions and highlight statictea code.
  • parseNumber.nim — Parse an int or float number string.
  • processTemplate.nim — Process the template.
  • readjson.nim — Read json content.
  • regexes.nim — Perl regular expression matching.
  • repl.nim — Run commands at a prompt.
  • replacement.nim — Handle the replacement block lines.
  • runCommand.nim — Run a command and fill in the variables dictionaries.
  • runner.nim — A standalone command to run Single Test File (stf) files.
  • signatures.nim — Statictea function signatures and parameter checking.
  • startingvars.nim — Get the starting variables.
  • statictea.nim — StaticTea; A template processor and language.
  • tempFile.nim — Create a temporary file.
  • timer.nim — Time how long it takes something to run.
  • unicodes.nim — Functions that deal with Unicode.
  • updateTemplate.nim — Update a template.
  • utf8decoder.nim — Functions for decoding and validating UTF-8.
  • variables.nim — Procedures for working with statictea variables.
  • vartypes.nim — StaticTea variable types.
  • version.nim — The current StaticTea version number.
