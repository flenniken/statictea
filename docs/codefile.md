# Found

The line endings found.<ul class="simple"><li>nothing = no special ending</li>
<li>plus = +</li>
<li>triple = """</li>
<li>newline = \n</li>
<li>plus_n = +\n</li>
<li>triple_n = """\n</li>
<li>crlf = \r\n</li>
<li>plus_crlf = +\r\n</li>
<li>triple_crlf = """\r\n</li>
</ul>
